## PCB
process control block   
프로세스에 대한 메타 데이터를 저장한 데이터   
프로세스가 생성되면 운영체제는 해당 PCB를 생성한다.   

프로그램 실행 -> 프로세스 생성 -> 스택 힙 등 구조를 기반으로 메모리가 할당   
-> 프로세스의 메타데이터들이 PCB에 저장되어 관리   

PCB는 일반 사용자가 접근하지 못하도록 커널 스택의 가장 앞부분에서 관리된다.   

### PCB의 구조 
프로세스 스케줄링 상태   
프로세스 ID   
프로세스 권한   
프로그램 카운터   
CPU 레지스터   
CPU 스케줄링 정보   
계정 정보   
I/O 상태 정보
